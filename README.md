# Coupon System

A Spring Boot REST API for validating and applying coupons.

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

## Features
- Validate Coupon
- Apply Coupon
- Prevent Duplicate Coupon Usage
- REST API

## API Endpoints

POST /validate-coupon

POST /apply-coupon

## How to Run

1. Clone the repository
2. Configure MySQL
3. Run the project
4. Test APIs using Postman
